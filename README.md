# A1W1 – Environment and AI Toolchain

This repository contains the **Week 1 capstone assignment** for the AI-assisted development track. It documents the setup of a modern JavaScript development environment, AI coding tools, and a Git workflow using **Conventional Commits**.

---

## 📌 Assignment Objectives

- Set up a professional development environment.
- Install and configure **Node.js (LTS)**, **Git**, and an AI-powered editor.
- Configure **Claude Code** or **Cursor** for AI-assisted development.
- Practice version control with **Conventional Commits**.
- Create a reusable project foundation for future assignments.

---

## 🛠️ Tech Stack

- **Runtime:** Node.js (LTS)
- **Language:** JavaScript (ES2023)
- **Version Control:** Git & GitHub
- **AI Assistant:** Claude Code / Cursor
- **Editor:** Cursor or Visual Studio Code
- **Package Manager:** npm

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/azijulhakimbd/A1W1-Environment-and-AI-toolchain.git
cd A1W1-Environment-and-AI-toolchain
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Verify Your Environment

```bash
node -v
npm -v
git --version
```
