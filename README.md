# A1W1 – Environment and AI Toolchain

[![Repository](https://img.shields.io/badge/GitHub-A1W1-blue?logo=github)](https://github.com/azijulhakimbd/A1W1-Environment-and-AI-toolchain)

This repository contains the **Week 1 capstone assignment** for the AI-assisted development track. It documents the setup of a modern JavaScript development environment, AI coding tools, and a Git workflow using [Conventional Commits](https://www.conventionalcommits.org/).

---

## 📌 Assignment Objectives

- Set up a professional development environment.
- Install and configure **Node.js (LTS)**, **Git**, and an AI-powered editor.
- Configure **Claude Code** or **Cursor** for AI-assisted development.
- Practice version control with **Conventional Commits**.
- Create a reusable project foundation for future assignments.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/azijulhakimbd/A1W1-Environment-and-AI-toolchain.git
cd A1W1-Environment-and-AI-toolchain
```

### 2. Install Dependencies

```bash
npm install
```

This reads `package.json` and prepares the project for future assignments. No runtime dependencies are required for Week 1.

### 3. Verify Your Environment

```bash
node -v
npm -v
git --version
```

You should see version numbers similar to:

```text
v22.x.x    # Node.js LTS
10.x.x     # npm
2.x.x      # Git
```

## 📝 Git Workflow

This repository follows the [Conventional Commits](https://www.conventionalcommits.org/) specification. Use a **type**, optional **scope**, and a short **description**:

```text
<type>: <description>
```

### Commit Types

| Type | When to use |
|------|-------------|
| `feat` | New features or initial project setup |
| `docs` | Documentation only |
| `fix` | Bug fixes or corrections |
| `chore` | Tooling, config, or maintenance |

### Example Commit Messages

```text
feat: initialize capstone repository
docs: create project README
docs: add CLAUDE instructions
fix: update repository documentation
```

> Use a space after the colon — `feat: message`, not `feat:message`.

### Basic Commands

```bash
git status
git add .
git commit -m "docs: update README with setup instructions"
git push
```

On first push, set the upstream branch if prompted:

```bash
git push --set-upstream origin main
```
